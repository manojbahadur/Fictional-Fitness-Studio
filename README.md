# Fitness Booking API

A Flask-based backend for managing fitness class bookings.

## Endpoints

- `GET /classes?timezone=...`
- `POST /book`
- `GET /bookings?email=...`

## Setup

```bash
pip install -r requirements.txt
python seed_data.py
python main.py
```
